Edit Node Values

Each node on the tree represents a record in the corresponding table. These records/nodes are added and edited using this form. It includes Name, Parent, Rank, Common Name, synonym information, Child Nodes and a Remarks field. Required fields include Name, Parent and Rank.

Nodes can not be moved using this form. To move a node drag and drop the node onto a new parent, then click on the Move button in the resulting dialog.
