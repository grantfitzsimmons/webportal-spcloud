Web Portal

Users with Admin privileges can enable/disable the exporting of live Web Links from fields that have been formatted as web links using the Preferences dialog:

  • For Mac: Choose Specify > Preferences > Web Portal.
  • For Windows and Linux: Choose Edit > Preferences > Web Portal.

Web Portal Preferences

  • Export WebLinks as URLs allows the values in all the web link fields to be exported as URLs, or live links. When the box is unchecked only the value in the field will be exported. For information about formatting a field as a Web Link visit Web Links

NOTE:  this feature is only operational for Web Links that have been formatted to use a single field. Web links configured to use more than one field will not be exported as URLs.
